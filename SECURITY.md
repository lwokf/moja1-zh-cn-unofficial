# Security policy

## Supported versions

Only the latest release is supported. Translation hooks are gated to the exact Moj A1 version listed in `MojA1Zh/config/runtime_rules.json`.

## Reporting a vulnerability

Use GitHub's private security advisory feature when possible. Do not open a public issue containing credentials, private account data, signing material, device logs or a working exploit.

Useful reports include unintended replacement of a number, amount, date or user-entered value; translation outside `hr.infinum.mojvip`; activation on an unsupported version; private text in logs; or an unexpected network permission/request.

## Root and LSPosed warning

LSPosed modules execute inside another application's process. Review the source, verify release checksums and use the module only on devices you control. No warranty is provided.
