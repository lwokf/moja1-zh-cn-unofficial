# Privacy

MojA1Zh is designed to work entirely on the device.

- The module requests no network permission.
- It does not upload UI text, account details, credentials, screenshots, logs, or device identifiers.
- It does not read or modify network requests or responses.
- It never translates text entered into `EditText` fields.
- Diagnostic logging records rule identifiers only, not matched source text or translated account content.

The repository intentionally excludes test-account screenshots, UIAutomator dumps, LSPosed databases, original APKs, device logs, and signing private keys.

When reporting a bug, remove or mask phone numbers, balances, e-mail addresses, voucher codes, dates, account identifiers and any other personal information.
